## Windows


解压后，双击 `start.bat` 即可运行。

![Alt text](/images/usage/image-3.png)


会在CMD中显示如下信息：

![Alt text](/images/usage/image-4.png)

上图选中的地方就是server的地址，复制到浏览器中打开即可。

(理论上会自动打开浏览器，但是有时候会失败（例如被杀软拦截），所以建议手动打开)
