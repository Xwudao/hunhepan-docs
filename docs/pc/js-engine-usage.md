# JS 引擎说明（PC 端）

PC 端的 js 引擎，支持全部 es5 语法，以及大部分 es6 语法；

> 但是由于 Android 端的 JS 引擎，仅支持 es5 语法，所以在开发过程中，需要注意，尽量不要使用 es6 语法，这样不同端的规则迁移，会更加方便；
