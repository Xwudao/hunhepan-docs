# ReMan-Lite 网盘搜索引擎程序

何为 ReMan-Lite？

`ReMan-Lite` 是一款拥有后台的个人网盘资源管理程序，和ReMan不同，ReMan-Lite吸取了ReMan的优点，同时主要是为了用户可以自行修改前台界面，以及SEO友好程度更高。

和 ReMan 的关系？

ReMan-Lite 是 ReMan 的轻量版，两者除了功能上有类似的地方，但却是两个独立的程序；
